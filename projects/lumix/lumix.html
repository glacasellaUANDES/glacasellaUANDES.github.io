DESCRIPTION

A nice photo organizer and editor
with many functionalities,using IBM Watson


A photo organizer and editor for Microsoft Windows, made with OOP C# WinForms. On this program, you will have your user and password, and you will be able to save your favorite pictures on it.

You can add tags to your photos, tag friends or ask IBM Watson what tag should you add to your picture. You can access your photos metadata, add a calification or filter your photos using the SearchBar by any tag or property.

With this program, you can also add filters to your photos (sepia, mirror, grayscale, old film, and many more) and you can use awesome features like adding censorship, create a collage, presentations, slideshow and many more.

WHAT I USED


  • C# OOP on .NET Framework
  • .NET Framework's WinForms
  • Visual Studio IDE
  • IBM Watson Developer Cloud v3.0 API with Pictures Recognition Service
