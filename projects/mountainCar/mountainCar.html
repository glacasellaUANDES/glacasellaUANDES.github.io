AI: QLEARNING
ON OPEN AI ENVIRONMENT

DESCRIPTION

Applying QLearning algorithm to teach an agent to drive up the mountain


On this project, I trained an AI Agent with Model-free Reinforcement Learning (QLearning) on the OpenAI's MountainCar-v0 environment. The main goal is to drive up the mountain on the right; however, the car's engine is not strong enough to scale the mountain in a single pass. Therefore, the only way to succeed is to drive back and forth to build up momentum.

In order to make the Agent succeed, I used Markov Processes and the Bellman Equation to make the state-action pairs quality update, and tryed with Greedy, ε-Greedy and ε-Descendant-Greedy algorithms

WHAT I USED


  • Python 3.5
  • Matplotlib
  • Numpy
  • OpenAI's Gym
